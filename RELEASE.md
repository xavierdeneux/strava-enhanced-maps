# Chrome

Make a zip that contains all the files
Inside this folder run: `zip -r chrome.zip ./*`

Send it in the dashboard developer:
https://chrome.google.com/webstore/devconsole/ 

# Firefox

Make a zip that contains the extension folder that contains all the files
Go on the parent folder and run `zip -r firefox.zip ./*`

Send it in the dashboard developer:
https://addons.mozilla.org/fr/developers/addons
